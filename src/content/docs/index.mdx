## What is AutoVio?

AutoVio is an **AI-powered video generation platform** that automates the full video production workflow. Create promotional and marketing videos by either:

- **Style Transfer** — Analyze a reference video and recreate it with new content while keeping the visual style
- **Content Remix** — Generate entirely new video content from text descriptions

## Key Features

- **Text to video generation** — Create videos from text prompts
- **Reference video analysis** — Analyze existing videos with vision AI to replicate styles
- **Scene-by-scene scenario generation** — LLM-powered scriptwriting
- **AI image generation** — Generate visuals per scene (Gemini, DALL-E)
- **Image to video conversion** — Animate images to clips (Veo, Runway)
- **Timeline editor** — Text overlays, image overlays, transitions, export
- **Template system** — Reusable overlay compositions for branding
- **REST API** — Full OpenAPI for integration
- **MCP server** — Use with Claude Desktop, Cursor, and other MCP clients
